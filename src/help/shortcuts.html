Keyboard shortcuts

  • Ctrl+z: Undo

Class keyboard shortcuts

  • Ctrl+A: Add a new attribute to the class
  • Cmd+A (On Mac): Add a new attribute to the class
  • Ctrl+P: Open the class' properties dialog box
  • Cmd+P (On Mac): Open the class' properties dialog box
  • Enter: Confirm changes when editing text in a class
  • Delete: Delete the class
